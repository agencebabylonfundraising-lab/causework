# CauseWork

> Le job qui compte. Vraiment.

## About
CauseWork is a job board for impact-driven careers in France, focused on Gen Z (18-28). BABYLON's founding partner — provides a proprietary pipeline of motivated fundraisers.

## Business Model
- Free job listings (launch phase)
- Premium featured listings: €199/month
- Employer branding packages: €499/month
- BABYLON uses it as proprietary recruitment pipeline

## Tech Stack
- Frontend: HTML/CSS/JS (migrating to Next.js)
- Backend: Node.js + Express (planned)
- Database: PostgreSQL (planned)

## Status
🚧 MVP built — backend + user accounts pending
